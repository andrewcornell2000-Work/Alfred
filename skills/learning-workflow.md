# Learning Workflow (skill pointer)

Full playbook: **`docs/LEARNING-WORKFLOW.md`**

Use when teaching Alfred a new tool, MCP, skill, or catalog entry.

## Quick rules

1. Read `docs/ALFRED-STRUCTURE.md` and `requirements/catalog-index.json` first
2. **Update existing** skill before creating a new one
3. Web search: 1–3 targeted queries on DISCOVER missions only
4. One deliverable per session with **"Try asking:"** examples
5. Run `python .github/scripts/validate_catalog.py` before commit
