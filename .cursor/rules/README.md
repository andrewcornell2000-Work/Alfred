# Cursor rules in this folder

**Canonical source:** [`cursor/rules/`](../cursor/rules/)

This workspace copy is synced from `cursor/rules/` so Cursor loads rules in-repo.
After editing rules, run:

```powershell
powershell -File scripts/sync-cursor-rules.ps1
```

Global rules are also provisioned to `~/.cursor/rules` by `Provision-Cursor.ps1`.
